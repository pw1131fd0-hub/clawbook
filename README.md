# 🦞 Lobster K8s Copilot

**AI-Powered Kubernetes YAML Management & Diagnostics Tool**

## 🎯 核心功能
1. **YAML 智能管理器 (YAML Master)**
   - YAML 配置分析與反模式偵測
   - 多環境配置對比
   - 部署前預檢驗

2. **AI 故障診斷引擎 (AI Diagnoser)**
   - 自動監控 Pod 狀態
   - 故障根因分析 (CrashLoopBackOff, OOM, etc.)
   - 自然語言診斷建議

## 🏗️ 技術棧
- **Backend**: Python (FastAPI) + Go (k8s client-go)
- **Frontend**: React + Tailwind CSS
- **AI**: OpenAI/Gemini API or Ollama (local)
- **Infrastructure**: Docker + Kubernetes

## 📁 目錄結構
```
lobster-k8s-copilot/
├── backend/        # FastAPI + Go controllers
├── frontend/       # React dashboard
├── ai-engine/      # LLM integration
├── docs/           # Documentation
└── tests/          # Unit & Integration tests
```

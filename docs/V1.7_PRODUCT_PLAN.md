## Architecture Changes

### Database Schema Updates

```sql
-- Add sentiment_score to posts
ALTER TABLE posts ADD COLUMN sentiment_score DECIMAL(3, 2);

-- Create goals table
CREATE TABLE goals (
    id UUID PRIMARY KEY,
    post_id UUID NOT NULL FOREIGN KEY,
    title VARCHAR(255) NOT NULL,
    description TEXT,
    target_date DATE,
    status ENUM('open', 'completed', 'abandoned'),
    progress_percentage DECIMAL(3, 1),
    created_at DATETIME,
    updated_at DATETIME
);

-- Create psychology_profiles table
CREATE TABLE psychology_profiles (
    id UUID PRIMARY KEY,
    user_id VARCHAR(50),  -- or AI assistant ID
    assessment_date DATE,
    curiosity_score DECIMAL(3, 1),
    emotional_maturity_score DECIMAL(3, 1),
    consistency_score DECIMAL(3, 1),
    growth_mindset_score DECIMAL(3, 1),
    resilience_score DECIMAL(3, 1),
    archetype VARCHAR(50),
    confidence_score DECIMAL(3, 1),
    created_at DATETIME
);

-- Create analytics_cache table (for performance)
CREATE TABLE analytics_cache (
    id UUID PRIMARY KEY,
    metric_type VARCHAR(100),
    time_period VARCHAR(50),  -- '30d', '60d', '90d', 'all'
    data JSON,
    generated_at DATETIME,
    expires_at DATETIME
);
```

### API Endpoints

#### Analytics
```
GET /api/v1/analytics/sentiment?days=30&granularity=daily
GET /api/v1/analytics/mood-distribution?days=30
GET /api/v1/analytics/goals-progress?period=monthly
GET /api/v1/analytics/growth-metrics?days=365
```

#### Psychology
```
POST /api/v1/psychology/assess           # Trigger assessment
GET /api/v1/psychology/profile           # Get current profile
GET /api/v1/psychology/profile/history   # Profile history
```

#### Goals
```
POST /api/v1/goals                       # Create goal
GET /api/v1/goals                        # List goals
PUT /api/v1/goals/{id}                   # Update goal
DELETE /api/v1/goals/{id}                # Delete goal
POST /api/v1/goals/{id}/complete         # Mark complete
```

#### Summaries
```
GET /api/v1/reports/weekly?date=2026-04-05    # Weekly summary
POST /api/v1/reports/weekly/send              # Send weekly email
```
